Relayer Network
Community Node Architecture
Anyone can run a relayer. The Gateway Contract auto-approves based on verifiable criteria. No Bpesa team member needs to manually whitelist anything.
RELAYER LIFECYCLE
STEP 1
Operator Registers
Calls registerRelayer() on Gateway Contract. Stakes minimum bKES. Provides server endpoint + payout stable preference.
STEP 2
Automated Checks Run
Backend service verifies: bKES stake · ETH balance · image hash · health endpoint reachable · 72hr minimum age · one wallet per relayer.
STEP 3
Approved + Subdomain Assigned
Gateway Contract emits RelayerApproved. DNS automation creates relayerN.node.bpesa.net. TLS cert auto-provisioned.
STEP 4
Live — Earning Rewards
Added to approved list. Mobile polls and starts routing txs. Weekly score tracked. ETH reimbursement + stable rewards accumulate.
STEP 5 (if needed)
Auto-Suspend + Reinstate
Health drop or ETH low → auto-suspended. Operator fixes issue → reinstatement loop re-runs checks → back online automatically.
SYBIL RESISTANCE STACK
Layer 1 — Economic Cost
bKES Stake Required
Real capital locked to get approved. Makes spinning up N fake nodes expensive. Slashable on repeated violations.
Layer 2 — Time Cost
72hr Minimum Age
Throwaway nodes can't earn. Must be registered and alive for 3 days before rewards kick in.
Layer 3 — ETH Floor
Minimum ETH Balance
Must maintain ETH balance to stay approved. Fake nodes that don't stay funded get auto-suspended.
Layer 4 — Image Integrity
Approved Image Hash On-chain
Modified Docker images detected and rejected. Gateway stores approved hash. Relayer self-reports on startup.

Keeper System
The Autonomous Brain
Three continuous loops run everything. Primary + secondary failover. A watchdog on a separate server. On-chain heartbeat for public verifiability.
THREE KEEPER LOOPS
Loop 1
Health Monitor
Every 60s
Pings every approved relayer /health endpoint. Checks ETH balance. Verifies image hash.
3 misses → degraded 5 misses → suspend
Loop 2
Reinstatement Processor
Every 10min
Re-runs full approval checks on all suspended relayers. Auto-reinstates on pass. DNS restored automatically.
Pass → reinstate Fail → stay suspended
Loop 3
Vault Health Monitor
Every hour
Checks Vault 2 ETH balance and Vault 3 stable balances. Triggers top-ups via LP conversion when below 1-week float.
Auto top-up Self-funded from Vault 1
